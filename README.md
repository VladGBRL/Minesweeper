Minesweeper Java Swing Project

This project is a desktop implementation of the classic Minesweeper game, developed in Java using Swing. While it preserves the core gameplay mechanics—revealing safe cells, flagging suspected mines, and clearing the board—it places a strong emphasis on user interface design and overall presentation.

Features
Classic Minesweeper gameplay
Multiple difficulty presets
Live mine counter and timer display
Custom graphical user interface
Sound effects and ambient audio
Side visuals and video elements
Victory and explosion animations
Additional Functionality

In addition to the gameplay, the project includes a utility module that implements Run-Length Encoding (RLE) for board state data. This is used for:

Debugging purposes
Analyzing and compressing board state information during gameplay
Project Scope

This project was developed as a course-style application, combining multiple concepts:

Game logic implementation
GUI design with Java Swing
Multimedia integration (audio and video)
Basic data compression techniques
